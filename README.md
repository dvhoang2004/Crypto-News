# Crypto Web App

## Giới thiệu
Đây là một ứng dụng web về Crypto được xây dựng với ReactJS, sử dụng API từ CoinGecko để hiển thị thông tin của các loại tiền điện tử. Ứng dụng có hai trang chính:

- **Trang Home**: Hiển thị thông tin chi tiết về các loại tiền điện tử, bao gồm giá trị hiện tại, biến động giá, và các chỉ số khác.
- **Trang Blog**: Cung cấp các bài viết, tin tức liên quan đến Crypto, giúp người dùng cập nhật các thông tin mới nhất trong thế giới tiền điện tử.

## Các tính năng chính
- **Hiển thị giá trị và thông tin của các loại Crypto**: Ứng dụng sử dụng API của CoinGecko để lấy thông tin chi tiết về giá trị, sự thay đổi của các loại tiền điện tử phổ biến.
- **Tìm kiếm nhanh các loại Crypto**: Người dùng có thể tìm kiếm nhanh chóng các loại tiền điện tử để xem các thông tin chi tiết của chúng.
- **Xem các bài viết về Crypto**: Trang Blog cung cấp các bài viết, phân tích, và tin tức về thị trường Crypto, giúp người dùng nắm bắt xu hướng mới.

## Công nghệ sử dụng
- **Frontend**: ReactJS (với các thư viện hỗ trợ như React Router, Axios, v.v.)
- **API**: CoinGecko API (cung cấp dữ liệu về Crypto)
- **Blog Content**: Có thể được quản lý qua một CMS hoặc dữ liệu tĩnh từ file markdown.
